
This is a fork of
[awesome-d3](https://github.com/wbkd/awesome-d3)
with just the github repos that use
[React](https://github.com/facebook/react).

I will also add in some personal commentary about each repo over time.  Initially all I will do is get all of the references for React and then as I have time add in my commentary.

## Awesome D3

 [![Awesome](https://cdn.rawgit.com/sindresorhus/awesome/d7305f38d29fed78fa85652e3a63e154dd8e8829/media/badge.svg)](https://github.com/sindresorhus/awesome)

##### Third Party Rocks

- [nivo](https://github.com/plouc/nivo) - Dataviz components for React with isomorphic ability

- [react-stockcharts](http://rrag.github.io/react-stockcharts) - Highly customizable stock charts

- [react-vis](https://github.com/uber-common/react-vis) - A collection of react components **built by Uber** to render common data visualization charts

- [recharts](http://recharts.org/) - Re-designed charting library built with React

- [semiotic](https://github.com/emeeks/semiotic) - A data visualization framework combining React & D3

- [victory](https://github.com/FormidableLabs/victory) - A collection of composable React components for building interactive data visualizations

- [vx](https://github.com/hshoff/vx) - Visualization components for React

##### Third Party

- [numberpicture](https://github.com/finnfiddle/number-picture/) - Collection of React components for composing visualizations [72 stars]

- [react-d3-components](https://github.com/codesuki/react-d3-components) - D3 Components [Still dependent on D3 version 3]

##### Third Party Repos that are no longer actively maintained

- [d3act](https://github.com/AnSavvides/d3act) - d3 with React **september 2016**
- [react-d3](https://github.com/esbullington/react-d3) - Charts for React **april 2016**
- [react-d3-library](https://github.com/react-d3-library/react-d3-library) - Library that allows developers the ability to use D3 in React **march 2017**
- [React D3 Map](https://github.com/react-d3) - Library For Building Composable And Declarative Maps **not maintained**
